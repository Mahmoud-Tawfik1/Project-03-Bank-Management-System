# 🏦 Bank Client Management System (C++)

A console-based **Bank Client Management System** built using **C++**.  
The application allows users to manage bank client records through a menu-driven interface while storing data permanently using text files.

This project focuses on practicing **structured programming**, file handling, and building a complete CRUD application.

## ✨ Features

🔹 Display all registered clients  
🔹 Add new client records

🔹 Update existing client information  
🔹 Delete client records  
🔹 Store data permanently using text files

## 🕹️ How the System Works

1. The program loads client data from a text file.
2. A menu is displayed to the user.
3. The user selects one of the available operations.
4. The requested operation is performed.
5. Any changes are saved automatically back to the file.
6. The program continues until the user chooses to exit.
